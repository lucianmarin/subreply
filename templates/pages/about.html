Web app install

  • Install on Android by adding it to home screen from Chrome
  • Install on iPhone or iPad by adding it to home screen from Safari
  • Similarly can be installed on Windows, Linux and macOS

Tech stack and donations

  1. High frequency Vultr VPS running on Ubuntu 24.04 LTS
  2. Coded in Python 3.12 using Falcon and Django ORM
  3. Speed up by Gunicorn in front of Nginx, PgBouncer in front of PostgreSQL
  4. Icons provided by Heroicons, Route 159 font by Sora Sagano
  5. Donate with PayPal to keep it ad-free and running indefinitely

Features and limitations

  1. Reverse-chronological order for all feeds
  2. Unique name handles like {{ emo.full_name | emojize }}
  3. Unique conversation view similar to a chat interface
  4. Light and dark themes based on OS preference
  5. Supports links, mentions, hashrefs and hashtags
  6. Limited to ASCII characters and emojis
  7. Limited to 640 characters per reply
  8. Limited to one link, mention, hashref or hashtag per reply
  9. No support for paragraphs, keeps conversation tidy
  10. No popularity counters or instant gratification

Keeping things private

  1. Edit or delete your replies before they are replied to
  2. Delete received replies before they are replied to
  3. No redistribution of any data, private or public based on GDPR
  4. Straightforward and complete account deletion
  5. Data export in a human readable format
